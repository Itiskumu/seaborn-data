# seaborn-data
My seaborn online data set

## **winequality-red.csv** 
红酒数据集。
该数据集为红酒通用数据集。包括1599个样本，11个红酒的理化性质，以及红酒的品质（评分从0到10）。

12个特征分别为：

```fixed acidity``` - 固定酸度

```volatile acidity``` - 挥发性酸度

```citric acid``` - 柠檬酸

```residual sugar``` - 残留糖

```chlorides``` - 氯离子

```free sulfur dioxide``` - 游离二氧化硫

```total sulfur dioxide``` - 总二氧化硫

```density``` - 密度

```pH``` - 酸碱度

```sulphates``` - 硫酸盐

```alcohol``` - 酒精度数

```quality``` - 品质
